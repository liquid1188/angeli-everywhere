# angeli.everywhere

Personal site for **Angeli** (Angel / Angie) — Filipino freelancer, virtual assistant, and travel storyteller.

> Quiet operations. Cinematic storytelling. Brand work that doesn't shout.

## About

A single-page personal site built around Angeli's brand voice: diary-like, poetic, warm. Muted color palette (paper, clay, terracotta, ink), editorial typography (Italiana display + Fraunces serif + Inter Tight), and a layout that prefers atmosphere to advertisement.

## Sections

- **About** — a working diary
- **Craft** — six core skills, practiced
- **Journal** — recent projects and what's at the desk
- **Off the clock** — interests beyond the work
- **Contact** — email, socials, freelance platforms

## Stack

Pure HTML and CSS. No build step. No JavaScript framework. A small intersection-observer script for scroll reveals. Hosted on GitHub Pages.

## Local preview

Open `index.html` in any browser. That's it.

## Deployment

This site is configured for GitHub Pages. Pushes to `main` deploy automatically once Pages is enabled in the repo settings (Settings → Pages → Deploy from branch → `main` / root).
